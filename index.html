mysql中文是？？？的情况

发表于 2018-01-17 |

问题描述

往数据库中提交数据（如注册用户）时，eclipse直接syso出的没问题，但数据库中该显示中文的地方全都是？？？？

解决方法

查阅资料后发现要在命令行改下面的编码，但是每次改完重启mysql后就又变回去了，最终得知需要在mysql安装路径修改my.ini文件。

  1. 命令行
  2. mysql
  3. show variables like ‘characterset%’;

  4. 发现很多不一样，必须修改为utf8

    set character_set_client=utf8 set character_set_connection=utf8; set character_set_database=utf8; set character_set_results=utf8; set character_set_server=utf8; set character_set_system=utf8; set collation_database=utf8_general_ci; set collation_connection=utf8_general_ci;

  1. 修改my.ini在[mysqld]中增加

    ‘character_set_server=utf8
    init_connect=’SET NAMES utf8’
